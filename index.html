Política de Privacidade

Última atualização: 2026

Esta Política de Privacidade descreve como o aplicativo EWIGNS coleta, utiliza e protege as informações dos usuários, em conformidade com a Lei Geral de Proteção de Dados (LGPD - Lei nº 13.709/2018).

1. Informações Coletadas

Podemos coletar as seguintes categorias de dados:

  • Dados de identificação: nome e e-mail fornecidos durante o cadastro/login
  • Dados de uso: interações dentro do aplicativo, participação em torneios e atividades
  • Dados técnicos: endereço IP, tipo de dispositivo, sistema operacional

2. Finalidade do Uso dos Dados

Os dados coletados são utilizados para:

  • Autenticação e gerenciamento de contas
  • Operação de funcionalidades como torneios e comunidade
  • Melhoria contínua do aplicativo
  • Prevenção de fraudes e segurança

3. Base Legal (LGPD)

O tratamento de dados é realizado com base nas seguintes hipóteses legais:

  • Execução de contrato (uso do aplicativo)
  • Consentimento do usuário
  • Legítimo interesse (melhoria do serviço)

4. Serviços de Terceiros

Firebase (Google)

Utilizamos o Firebase para autenticação, banco de dados e análise de uso. O Firebase pode coletar e processar dados conforme sua própria política de privacidade.

Publicidade (Google AdMob - futuro)

O aplicativo poderá exibir anúncios por meio do Google AdMob. Esses serviços podem utilizar identificadores de dispositivo e cookies para personalização de anúncios.

5. Compartilhamento de Dados

Não vendemos dados pessoais. O compartilhamento ocorre apenas:

  • Com provedores de serviço (ex: Firebase)
  • Para cumprimento de obrigações legais

6. Armazenamento e Segurança

Os dados são armazenados em ambientes seguros e protegidos. Adotamos medidas técnicas e organizacionais para evitar acessos não autorizados.

7. Retenção de Dados

Os dados são mantidos apenas pelo tempo necessário para cumprir as finalidades descritas nesta política, ou conforme exigido por lei.

8. Direitos do Usuário

Nos termos da LGPD, você tem direito a:

  • Acessar seus dados
  • Corrigir dados incompletos
  • Solicitar exclusão de dados
  • Revogar consentimento

9. Exclusão de Conta e Dados

O usuário pode solicitar a exclusão da conta e de todos os dados associados entrando em contato pelo e-mail abaixo.

10. Alterações nesta Política

Esta Política pode ser atualizada periodicamente. Recomendamos a revisão regular.
